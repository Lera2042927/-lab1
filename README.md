Лабораторная работа №1: Первичное исследование и оценка качества данных
Описание проекта
Первичный анализ датасета «500 Greatest Songs of All Time» в рамках дисциплины «Интеллектуальный анализ данных».

Предметная область: Музыкальные рейтинги, исторический анализ популярности песен.Возможные задачи ИАД:Анализ временных трендов в музыке, рекомендательная система для похожих песен, исследование влияния года выпуска на позицию в рейтинге.
Цель лабораторной работы: аудит качества данных, построение паспорта датасета и выявление проблем.

Структура проекта
notebooks/01_data_understanding.ipynb— основной блокнот Jupyter с анализом
report/quality_report.md— текстовый отчёт с выводами и улучшение схемы
data/— установка даты или загрузка скрипта
README.md — краткое описание проекта и инструкции по запуску. 

Инструкции по запуску
Установите в зависимости:
pip install pandas numpy matplotlib seaborn jupyter
